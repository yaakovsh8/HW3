## תיאור הפרויקט
פרויקט זה הוא מערכת מבוססת Flask להצגת שאלות ותשובות, כולל יכולת חיפוש, הצגת תמונות תשובות בצורה אינטראקטיבית ועיצוב מותאם למשתמש.

## דרישות מערכת

- Flask 2.0+
- דפדפן אינטרנט מודרני
